## 网站路由表

| 路由 | 类型 | 说明 |
|------|------|------|
| `/` | 静态 | 首页（市场切换 + 自选 + 搜索 + 分类） |
| `/stock/[symbol]` | 动态 | 股票详情（支持美股和 .HK 港股） |
| `/api/stocks/trending?market=` | API | 热门股票（us/hk） |
| `/api/stocks/search?q=` | API | 股票搜索 |
| `/api/stocks/quotes?symbols=` | API | 批量报价 |
| `/api/stock/[symbol]/history` | API | 历史K线数据 |
| `/api/stock/[symbol]/news` | API | 新闻聚合 |
| `/api/stock/[symbol]/analysis` | API | AI 分析 |
| `/api/comments` | API | 留言 CRUD |
