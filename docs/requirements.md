# 美股分析网站 — 需求文档

## 项目概述

构建一个美股分析网站，用户可通过浏览器直接使用。提供股票总览、价格走势、K线分析、新闻聚合、AI 智能分析和留言讨论功能。

## 目标用户

对美股投资感兴趣的普通用户，无需任何技术背景。

## 功能需求

### F1. 首页 — 股票总览
- 展示热门美股列表（默认显示市场热门股票）
- 每只股票显示：股票代码、公司名称、当前价格、涨跌幅
- 支持按分类切换：热门股票 / 涨幅最大 / 交易最活跃
- 支持股票搜索（输入代码或公司名称自动补全）
- 点击股票卡片进入详情页

### F2. 股票详情页
- 展示股票核心信息：代码、公司名、当前价格、涨跌额、涨跌幅
- 展示关键指标：成交量、市值、52周最高/最低

### F3. 价格走势图
- 折线图展示历史价格变化
- 默认展示近 1 个月数据
- 支持切换时间范围

### F4. K线图
- 蜡烛图展示 OHLC 数据（开盘、最高、最低、收盘）
- 附带成交量柱状图
- 支持切换周期：日K / 周K / 月K
- 图表支持缩放和拖拽

### F5. 新闻聚合
- 展示该股票相关的最新新闻
- 每条新闻显示：标题、来源、发布时间、摘要
- 点击可跳转到原文

### F6. AI 分析
- 汇总该股票的相关新闻
- AI 给出情绪判断：看涨 / 看跌 / 中性
- 提供简短的文字分析总结
- 标注分析方法和可信度

### F7. 留言讨论
- 每只股票下方有留言区
- 无需登录，游客可直接留言
- 输入昵称（可选，默认为"匿名用户"）和留言内容
- 留言按时间倒序排列
- 支持删除留言（简单管理）

## 非功能需求

- **界面语言**：中文
- **数据来源**：美股（NYSE、NASDAQ）
- **费用**：全部使用免费 API
- **性能**：页面首次加载应在合理时间内完成
- **兼容性**：支持主流浏览器（Chrome、Safari、Edge）
